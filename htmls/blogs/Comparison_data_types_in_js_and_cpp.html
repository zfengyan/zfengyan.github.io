Comparison - data types in JavaScript and C++

In JavaScript there are 8 fundamental data types: INTRODUCTION TO JAVASCRIPT - Data Types.

Let's see for an example: 


	// Number
	let myNumber = 42; // an integer
	let myDecimal = 3.14; // a floating-point number
	
	// BigInt
	let myBigInt = 12345678901234567890n;
	
	// String
	let myString = 'Hello, world!';
	
	// Boolean
	let isTrue = true;
	let isFalse = false;
	
	// Null
	let myNull = null;
	
	// Undefined
	let myUndefined;
	
	// Symbol
	const mySymbol = Symbol('mySymbol');
	
	// Object
	let myObject = {
		name: 'John',
		age: 30,
		city: 'New York'
	};
